CS622 Assignment 4

This assignment compared SGDClassifier and GaussianNB for classifying raisin varieties from physical measurements.

Demonstrated Skills

  • Data preprocessing and scaling
  • Model comparison using accuracy, F1, sensitivity, and log loss
  • Model selection for balanced and probabilistic performance

Project Summary

  • Standardized features with StandardScaler and encoded the target with LabelEncoder.
  • Tested both models on an 80/20 train-test split and evaluated their balance across class predictions.
  • Found that the SGDClassifier provided more stable, well-calibrated predictions while GaussianNB favored sensitivity.

Overall Project Summary

The project highlighted how different classifiers can excel on different metrics. It reinforced the importance of selecting a model based on the business or analytical goal rather than relying on accuracy alone.
