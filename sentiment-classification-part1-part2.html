Sentiment Classification (Part 1 & Part 2)

This two-part project explores sentiment classification across two text environments: short, noisy social-media posts and longer review-style documents.

Demonstrated Skills

  • Text cleaning, tokenization, and feature preparation
  • Word2Vec embedding training and document-level vectorization
  • Binary sentiment modeling with logistic regression and cross-validation

Project Summary

  • Part 1 focused on a large Twitter-style dataset and showed that a simple embedding-plus-logistic-regression pipeline can reach strong baseline accuracy.
  • Part 2 moved to longer IMDb-style reviews and demonstrated how richer context improves semantic stability.
  • Both parts highlighted that text length, embedding quality, and sequence context influence model robustness.

Overall Project Summary

The combined work shows the value of building a solid NLP pipeline from the ground up. It also reveals how the same modeling approach behaves differently across short- and long-form text, making the results useful for both practical classification work and deeper language-analysis discussions.
