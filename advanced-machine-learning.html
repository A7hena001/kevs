Advanced Machine Learning

This project brings together neural-network experimentation and an applied AI concept for identifying sensitive information in unstructured business documents.

Demonstrated Skills

  • Feedforward neural networks and convolutional neural networks
  • PyTorch model development and training workflow
  • Applied AI design for privacy and document screening

Project Summary

  • Compared a dense neural network with a CNN for handwritten-digit classification using the USPS dataset.
  • Achieved strong cross-validation accuracy, with the CNN showing a slight edge because of its spatial feature extraction.
  • Extended the concept into a teach-back presentation for building a custom ANN pipeline for detecting PII in documents.

Overall Project Summary

These efforts demonstrate a strong progression from foundational deep learning to applied, security-focused AI. The work highlights how neural networks can move from classical image classification into high-value workflows that support privacy protection and document governance.
