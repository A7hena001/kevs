CS622 YOLO

Computer Vision / Deep Learning

This project compares YOLOv5 and YOLOv8 for aerial car detection and highlights how model architecture, training stability, and hardware constraints shape practical performance.

Demonstrated Skills

  • Object detection model comparison and benchmarking
  • Training diagnostics using precision, recall, and mAP
  • Evaluation of model stability and deployment feasibility

Project Summary

  • Trained YOLOv5 and YOLOv8 variants on a single-class aerial dataset of parking-lot vehicles.
  • Measured 18 training combinations across several epochs and model sizes.
  • Observed that YOLOv5 achieved smoother convergence and better generalization in this setting.

Overall Project Summary

The analysis showed that newer architecture does not always guarantee better results. With limited data and constrained hardware, YOLOv5 delivered more reliable performance, while YOLOv8 demonstrated strong potential but required more careful tuning.
