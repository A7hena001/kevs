Relational Database Management & Data Quality Analysis

This project focused on building a trustworthy data foundation for franchise and character analysis, while quantifying where missing information could mislead the story.

Schema Design Created structured relationships for reliable querying.
Data Hygiene Measured gaps like missing gender and wand-core values.
Quality Insight Showed how incomplete data can distort analytical conclusions.

What I built

  • Designed relational tables and links between entities.
  • Audited the database for missing or inconsistent values.
  • Documented how quality issues affect downstream analysis.

Why it matters

  • Improved confidence in the resulting analysis.
  • Made data quality visible rather than assumed.
  • Created a more reliable base for future reporting.
