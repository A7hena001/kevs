Cloud & NLP Deployment

This project combines cloud deployment and natural language processing into a scalable sentiment analysis system for short-form and mixed-emotion text.

Demonstrated Skills

  • Cloud-native API design with AWS Lambda and API Gateway
  • Natural language preprocessing and sentiment labeling
  • Model evaluation and comparison across cloud and custom ML approaches

Project Summary

  • Built a serverless sentiment-analysis microservice that accepted text input and returned structured sentiment categories.
  • Prepared a blended dataset and balanced class distributions to reduce the bias seen in early model iterations.
  • Benchmarking included GPT-4, Grammarly, AWS Comprehend, and a custom multinomial logistic-regression model.

Overall Project Summary

This work connected scalable cloud deployment with practical NLP analysis. The final system showed that cloud-hosted services can deliver efficient, production-style sentiment classification while also revealing the trade-offs between speed, nuance, and contextual understanding.
