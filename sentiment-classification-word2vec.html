Natural Language Processing

Sentiment Classification Pipeline with Word2Vec

This project developed a complete NLP pipeline that turned raw text into sentiment predictions using Word2Vec embeddings and logistic regression.

50K Reviews Trained on a large IMDb review dataset.
88% Accuracy Reached strong classification performance after tuning.
Cross-Validation Used 5-fold validation to check robustness.

Pipeline

  • Preprocessed and cleaned review text.
  • Created skip-gram Word2Vec embeddings.
  • Trained a logistic regression classifier.

Result

  • Delivered a reliable sentiment classification workflow.
  • Established a strong baseline for future NLP experiments.
  • Demonstrated practical text classification on real data.
