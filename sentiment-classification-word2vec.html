Sentiment Classification Pipeline with Word2Vec

This NLP project turns review text into meaningful sentiment signals using word embeddings and a lightweight classification model.

50K Reviews Trained on a substantial IMDb dataset of review text.
88% Accuracy Reached strong sentiment discrimination after tuning.
Robust Testing Used 5-fold cross-validation to check consistency.

How the pipeline worked

  • Cleaned and prepared review text for analysis.
  • Built skip-gram Word2Vec embeddings to represent meaning.
  • Trained logistic regression to classify sentiment.

Why it stands out

  • Converted raw language into structured sentiment labels.
  • Showed how classic models can still perform well with rich embeddings.
  • Provided a practical path for text classification workflows.
