CS622 HW3 Linear and Ridge Regression

This assignment implemented linear and ridge regression from scratch and tested how regularization affects prediction quality.

Model Fit

Demonstrated Skills

  • Matrix-based regression implementation
  • Feature normalization and intercept handling
  • Hyperparameter tuning for ridge regularization

Project Summary

  • Prepared the Auto MPG dataset by removing nonessential columns and applying Z-score normalization.
  • Compared linear regression with ridge regression across multiple train-test splits and lambda values.
  • Observed that ridge regression improved stability in some splits while being sensitive to regularization strength.

Overall Project Summary

The project connected theoretical regression methods to practical implementation and showed that regularization is a powerful tool when feature relationships are strong but data splits are unstable.
