CS622 Assignment 3

This assignment used a Gaussian Naive Bayes classifier to identify raisin variety from geometric and structural features.

Demonstrated Skills

  • Multiclass-friendly data preprocessing for tabular features
  • Naive Bayes classification and evaluation
  • Interpretation of sensitivity, specificity, and F1-scores

Project Summary

  • Converted the Excel dataset to CSV format and encoded the target labels for modeling.
  • Split the data into training and testing sets and evaluated model behavior on unseen data.
  • Observed strong generalization with high sensitivity for one raisin class and balanced performance overall.

Overall Project Summary

The project demonstrated that a relatively simple probabilistic model can perform competitively on structured data when the feature set is well prepared and the task is appropriately scoped.
